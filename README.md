## Descripción

¡Hola! Soy Gabriela Riera y este es mi portfolio como diseñadora gráfica. Aquí te enseño mis proyectos de manera clara y minimalista, con un toque de animaciones sutiles.  He diseñado la web para que se vea bien en cualquier dispositivo, desde móviles hasta pantallas grandes.  

Mi objetivo con este portfolio es compartir mis trabajos de forma profesional, transmitiendo mi estilo y cuidando la experiencia del usuario con interacciones suaves y navegación intuitiva.

---

## Tech Stack

- HTML5 semántico
- CSS3 (variables, flexbox, grid, animaciones)
- JavaScript Vanilla
- Tipografía personalizada (Field Gothic XX Condensed)
- Diseño responsive para móviles, tablets y escritorio
- Animaciones CSS y hover states
- Prefers-reduced-motion para accesibilidad

---

## Instalación / Setup

Este proyecto es estático y fácil de abrir en cualquier navegador:

1. Clona el repositorio:

```bash
git clone https://github.com/gabrielaiguess/graphicdesignportfoliogabrielaiguess.git
```

2. Abre `index.html` en tu navegador.  

### Opcional: Previsualización en desarrollo

Si deseas usar un servidor local para ver cambios:

```bash
# Con VS Code y la extensión Live Server
code .
# Luego clic en "Go Live" para previsualizar el proyecto
```

## Funcionalidades principales

- Navegación fija en la parte superior para acceso rápido a secciones  
- Presentación ordenada de proyectos con estilo minimalista  
- Animaciones CSS suaves en transiciones y botones  
- Responsive: adapta layout en móviles, tablets y escritorios grandes  
- Prefers-reduced-motion implementado para usuarios que lo requieren  
- Hover states en elementos interactivos  
- Formulario de contacto funcional con labels correctamente asociados  
- Buen contraste de colores para accesibilidad (WCAG AA)  
- Optimización de imágenes y carga eficiente de fuentes
